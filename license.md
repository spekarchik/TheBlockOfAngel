# README (EN)

## Mod Name

Welcome! This mod was created by Sergey Pekarchik specifically for NeoForge 1.21.4+.

### Features:
- Fully original textures
- Supports Minecraft 1.21.4 and 1.21.5
- Unique gameplay mechanics (e.g., Burned Paper Block that attracts creepers and repels bees)

### Installation:
- Make sure you have Minecraft 1.21.4 or 1.21.5 with NeoForge installed
- Drop the mod into your `mods` folder
- Enjoy!
